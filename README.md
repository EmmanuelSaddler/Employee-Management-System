# Employee Management System

Welcome to the Employee Management System! This was a project that I had to do for college. What it essentially does is let you categorize your employees through a command prompt layout.

## Features
This program can:

Add and Remove Employees

Edit Existing Employees

Searching for Employees through SSN

Importing and exporting lists of employees

## Installation
To install, download the .py file from GitHub.
This program requires Python for use. Make sure its installed.
